4. Outlier Detection (IQR Method)
---------------------------------
- Outlier definition: Values below Q1 - 1.5*IQR or above Q3 + 1.5*IQR
- Features with significant outliers:
  - LotFrontage: 259
  - GrLivArea: 50
  - SalePrice: 28
- Outlier Flag:
  - Created 'Outlier_Flag' column: 0 = no outlier, 1 = at least one outlier
  - Rows with outliers: ~900
  - Rows without outliers: ~560

5. Outlier Handling
------------------
- Strategy:
  - Cap extreme values using IQR bounds (Winsorization) for numerical features to preserve all rows.
  - Alternatively, remove extreme outliers for very skewed features if necessary.
- Reason:
  - Prevent distortion of mean, std, and correlations while keeping dataset size sufficient for analysis.
